The **Internship Management System** is a full-stack web application designed to streamline the internship recruitment process. It provides a seamless interface for students to find opportunities and for specific user roles to manage applications.

The frontend is built with **React**, featuring a premium, glassmorphism-inspired design with rich gradients, smooth animations, and a fully responsive layout that looks great on any device.

## ğŸš€ Getting Started

Follow these steps to get a local copy running.

### Prerequisites

*   Node.js (v18+)
*   npm or yarn
*   Docker (Optional)

### Installation

1.  **Clone the repo**
    ```sh
     git clone https://github.com/Nourhenebenothmen22/internship-management-system.git
    cd internship-management-system
    ```

2.  **Install Frontend Dependencies**
    ```sh
    cd frontend
    npm install
    ```

3.  **Run the Application**
    ```sh
    npm run dev
    ```
    The app will be available at `http://localhost:5173`.
